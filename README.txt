# CookieTils

**CookieTils** is a Forge 1.8.9 **client-side utility mod** for **Hypixel SkyBlock**, designed to help with all aspects of gameplay — from dungeons to general quality-of-life improvements.

## ✨ Features

✅ Secret ESP for Dungeons (Chests, Levers, Bats)  
✅ Configurable colors, transparency, and toggles  
✅ Lightweight and optimized for performance  
✅ Future support planned for other SkyBlock areas (Foraging, Slayers, Kuudra, etc.)

## 📦 Installation

1. Install [Minecraft Forge 1.8.9](https://files.minecraftforge.net/)  
2. Download the CookieTils `.jar` from [Releases](https://github.com/SoulLessCookie/CookieTils/releases)  
3. Place it in your `.minecraft/mods` folder  
4. Launch Minecraft with Forge

> This is a **client-only mod** and safe to use on Hypixel.
I feel like I have to specify that this mod (as is), at no point breaks or will break hypixel rules or TOS.

## ⚙️ Configuration

- In-game config GUI available via `Mods → CookieTils → Config`
- Toggle features, change ESP colors, and adjust transparency
- Optional command support: Will be added later
