# CookieTils

**CookieTils** is a Forge 1.8.9 **client-side utility mod** for **Hypixel SkyBlock**, designed to enhance gameplay with useful tools and overlays — starting with dungeons, and expanding into other areas of the game.

## ✨ Features

✅ Dungeon Secret ESP (Chests, Levers, Bats)  
✅ Configurable colors, transparency, and feature toggles  
✅ Lightweight, clean, and performance-friendly  
✅ Designed for future expansion across SkyBlock (Foraging, Slayers, Kuudra, etc.)

## ⚠️ Disclaimer

This mod is **client-side only** and **does not break** Hypixel’s rules or Terms of Service.  
CookieTils does not automate gameplay, modify packets, or perform any unauthorized actions.

> Use responsibly and always double-check [Hypixel’s rules](https://hypixel.net/rules) if unsure.

## 📦 Installation

1. Install [Minecraft Forge 1.8.9](https://files.minecraftforge.net/)
2. Download the latest CookieTils `.jar` from the [Releases page](https://github.com/SoulLessCookie/CookieTils/releases)
3. Drop the file into your `.minecraft/mods` folder
4. Launch Minecraft with the Forge profile

## ⚙️ Configuration

- Open the in-game config via `Mods → CookieTils → Config`
- Toggle features, adjust ESP transparency, and set custom colors
- CLI command support (e.g. `/cookietils reload`) coming soon
